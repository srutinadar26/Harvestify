# 🌾 Harvestify Frontend

This is the frontend of Harvestify built using React. It provides an intuitive and responsive interface for farmers to access real-time insights and intelligent recommendations.

## 🚀 Features

- Real-time weather dashboard
- Live mandi market prices
- Multi-language support (English, Hindi, Marathi)
- Dark/Light mode
- Chatbot with offline support for basic queries
- Voice-enabled interaction
- Responsive UI for mobile, tablet, and desktop

## 🛠️ Tech Stack

- React
- Tailwind CSS
- React Router
- Context API
- Framer Motion
- Recharts
- i18next

## 📡 API Integration

- Weather API (real-time weather data)
- data.gov.in (market prices)
- Firebase Authentication

## ⚡ Run Locally

```bash
npm install
npm start

# 🌾 Harvestify Backend

This is the backend service for Harvestify, responsible for handling data processing, API integrations, authentication, and intelligent recommendations.

## 🚀 Features

- User authentication and authorization
- API integration for weather and market data
- Crop recommendation logic
- Chatbot response handling
- Data processing for frontend consumption

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Firebase Authentication

## 🔌 API Responsibilities

- Fetch and process weather data
- Fetch and filter mandi market prices
- Handle user requests and responses
- Serve recommendation results

```bash
npm install
node app.js
